# Geomagnetic-Storm-Risk-Prediction-using-XGBoost

A machine learning-based platform that analyzes the relationship between solar activity and geomagnetic disturbances on Earth. The system integrates historical space weather datasets, performs temporal correlation analysis, and predicts geomagnetic storm risk using XGBoost.


## Features

- Solar flare data collection and preprocessing
- Temporal alignment of solar and geomagnetic events
- Pearson and Spearman correlation analysis
- Lag-based cross-correlation analysis
- Geomagnetic Risk Score prediction using XGBoost
- Interactive Streamlit dashboard
- Risk visualization and impact assessment

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- SHAP
- Streamlit
- Matplotlib


## Data Sources

- NOAA GOES Solar Flare Dataset
- GFZ Potsdam Kp & Ap Indices
- WDC Kyoto Dst Index
